# AliceLily – Projektinstruktioner

## Arbetsflöde för uppladdning
1. Öppna index.html lokalt i webbläsaren
2. Fråga om den ser bra ut
3. Om OK: döp om befintlig index.html på GitHub till index_old.html
4. Pusha den nya index.html
5. Ge länk till GitHub Pages: https://vg1414.github.io/AliceLily/
